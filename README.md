## 有人说，真正的友谊就像是夜空中的微光——它可能不是时刻都如太阳般耀眼，但当你的世界陷入黑暗时，它一定会悄然亮起，为你指引方向。

真正的友谊，往往藏在这些触手可得的细节里：

无需面具的舒适： 在他面前，你不用刻意维持完美。你可以肆无忌惮地大笑，也可以毫无顾忌地流泪，甚至在长久的沉默中，也不会感到半分尴尬。

风雨同舟的默契： 完美的友谊不是从不争吵，而是吵过之后，只要一句“吃不吃夜宵”，就能瞬间破冰。它经得起时间的洗练，也熬得过距离的疏远。

顶峰相见的底气： 好的朋友，是在你低谷时拉你一把的暖手，更是你顶峰时为你真心喝彩的鼓掌者。你们各自独立，却又在精神上相互滋养。

世界上最美妙的事情之一，莫过于当你伸出手时，刚好有一只手坚定地握住它。那种感觉在告诉你：别怕，有我在。
